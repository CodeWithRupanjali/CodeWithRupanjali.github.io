# CodeWithRupanjali — Personal Portfolio

Premium personal brand website for **Rupanjali Kumari**.

## Files
- `index.html` — structure & content
- `style.css` — premium black / deep-red theme
- `script.js` — particles, nav, reveals, counters, FAQ, tabs, form
- `logo.jpg` / `logo.png` — official brand logo (do not alter)

## Local preview
Open `index.html` in a browser, or:

```bash
npx serve .
```

## Deploy (GitHub Pages)
1. Create a repo (e.g. `CodeWithRupanjali.github.io` or `portfolio`)
2. Push these files to the `main` branch
3. Settings → Pages → Deploy from `main` / root
4. Live URL will be under `https://codewithrupanjali.github.io/...`

## Resume
- `Rupanjali_Kumari_Resume.pdf` — professional one-page resume (portfolio-synced)
- Edit source: `resume.html` then regenerate PDF if needed
- Homepage **Download Resume** button downloads the PDF
